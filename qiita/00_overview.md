# OpsKitPro を作りました。DNS / IP / サイト診断をまとめた運用ツール集です

はじめまして。  
**OpsKitPro** という運用ツールサイトを作っています。

Qiita の最初の投稿として、まずは自分が何を作っているのかを簡単に紹介します。

OpsKitPro は、DNS / IP / サイト診断 / SSL / CDN / WebSocket など、運用や障害対応でよく使う確認作業をひとつにまとめたツール集です。

## 何を解決したいのか

サイトが開かないとき、確認したいものはだいたい決まっています。

- DNS は正しく引けているか
- IP はどこから来ているか
- HTTP は返っているか
- SSL は正常か
- CDN が前面にいるか

こうした作業を毎回別々のサイトで確認するのは、地味ですがコストが高いです。  
OpsKitPro は、その切り替えの手間を減らしたいと思って作りました。

## どんな工夫をしているか

### 1. 先に結論が見える
まず知りたいのは、細かいログよりも「今どうなっているか」です。  
そのため、結果は結論が先に見えるようにしています。

### 2. すぐ次の行動に進める
エラーが出たときに、ただ失敗を見せるだけでは足りません。  
「次にどこを見ればいいか」が分かるように、部分成功や補足情報も残しています。

### 3. 長文は知識ベースに逃がす
主サイトは軽く保ちたいので、長文の解説や実装メモは **kb.opskitpro.com** に分離しています。

OpsKitPro は、運用や障害対応でよく使う確認作業を、できるだけ速く、わかりやすく、ひとつの場所に集めることを目指して作っています。
