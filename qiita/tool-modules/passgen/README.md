**タイトル案:** `パスワード生成器 passgen を作った理由。必要なものをすぐ出せるようにしたかった`

- 何を解決したいか

- 1 ページで完結させたい理由

- 表示領域とコピー導線

- `crypto.randomUUID()` の使い方

- 履歴保存と再利用

**タイトル案:** `passgen の使い方。パスワード、UUID、PIN を素早く作る`
