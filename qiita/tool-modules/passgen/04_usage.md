# passgen の使い方。パスワード、UUID、PIN を素早く作る

passgen は、複雑な設定を覚えなくても使えることを前提にしています。
基本は「モードを選んで、条件を少し整えて、コピーする」だけです。

## よくある使い方

- サイト登録用に長めのパスワードを作る
- 一時的な確認用に短い PIN を作る
- DB やログの紐付けに使う UUID を作る
- 手元のテストデータを一瞬で埋める

## 迷いにくい使い方のコツ

1. まず用途を決める
2. 文字数やモードを最小限だけ調整する
3. 生成したらすぐコピーする
4. 必要なら履歴から再確認する

この順番にすると、画面を行き来せずに終わります。

## 運用での向き・不向き

passgen は、日常の小さな作業を早くするためのものです。
大規模なシークレット管理やチーム共有の仕組みの代わりではありません。
その代わり、"今この瞬間に必要な値を素早く作る" という用途ではかなり相性がいいです。

短い作業ほど、待ち時間や画面遷移が目立ちます。
passgen はその部分を減らすための道具として作っています。
