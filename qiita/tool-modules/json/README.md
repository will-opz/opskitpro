# json series

OpsKitPro の JSON ツール `json` を、課題・設計・実装・使い方に分けて整理します。

## 予定構成

### 1. 課題編
**タイトル案:** `JSON 整理器 json を作った理由。読みたいのは“構造”だった`

- 何を整理したいのか
- API 応答・設定・ログの共通課題
- 一覧ではなく作業台にした理由

### 2. 設計編
**タイトル案:** `json の UI 設計。編集・修復・変換を 1 ページにまとめる`

- エディタ中心にした理由
- 修復、変換、比較の分離
- ワークベンチとしての構成

### 3. 実装編
**タイトル案:** `json の実装。修復・変換・diff・schema を分割する`

- 修復ロジックの分離
- `jq` / schema / diff の分割
- hook と component の責務分担

### 4. 使い方編
**タイトル案:** `json の使い方。貼る、直す、比べる、抜き出す`

- 最短の利用手順
- 修復と整形の使い分け
- 実務での典型パターン
