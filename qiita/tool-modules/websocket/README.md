# websocket series

OpsKitPro の WebSocket 調試ツール `websocket` を、課題・設計・実装・使い方に分けて整理します。

## 予定構成

### 1. 課題編
**タイトル案:** `WebSocket デバッガーを作った理由。接続後の操作を楽にしたかった`

- 何が面倒だったのか
- 単純な接続確認だけでは足りない理由
- 複数接続が必要だった背景

### 2. 設計編
**タイトル案:** `websocket の UI 設計。接続・送信・ログを分けて見せる`

- 接続パネル
- メッセージ編集
- ログと統計の分離

### 3. 実装編
**タイトル案:** `websocket の実装。useWebSocket と複数接続の整理`

- `useWebSocket` の責務
- `useMultiConnection` の役割
- 履歴・テンプレート・バイナリ処理

### 4. 使い方編
**タイトル案:** `websocket の使い方。テキスト送信からバイナリ確認まで`

- 接続手順
- テンプレート送信
- 会話の保存と再利用
