# qrgen series

OpsKitPro の QR 生成ツール `qrgen` を、課題・設計・実装・使い方に分けて整理します。

## 予定構成

### 1. 課題編
**タイトル案:** `QR 生成ツール qrgen を作った理由。入力したらすぐ見えることを優先した`

- どんな場面で使うか
- 既存 QR ツールの不便さ
- 単純な入出力に絞った理由

### 2. 設計編
**タイトル案:** `qrgen の UI 設計。入力とプレビューだけに絞る`

- 2 段構成にした理由
- ダウンロード導線
- 装飾を減らした理由

### 3. 実装編
**タイトル案:** `qrgen の実装。SVG 生成から PNG ダウンロードまで`

- `QRCodeSVG` の利用
- SVG を PNG に変換する流れ
- ブラウザ内完結の構成

### 4. 使い方編
**タイトル案:** `qrgen の使い方。リンクや短文をすぐ QR にする`

- 入力からプレビューまで
- PNG 保存の流れ
- 共有用途の例
