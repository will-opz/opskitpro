## 🚀 核心功能增强 (Core Features)

- [ ] **全协议 DNS 审计**
  - [ ] 增加 MX (邮件交换)、AAAA (IPv6) 记录检测。
  - [ ] 增加 TXT 记录解析 (SPF/DKIM/DMARC 专项分析)。
  - [ ] 实现 CAA (认证机构授权) 记录检查。

- [ ] **硬核 SSL/TLS 安全分析**
  - [ ] 实现证书链 (Certificate Chain) 完整性校验。
  - [ ] 检测 HSTS、OCSP Stapling 开启状态。
  - [ ] 对 TLS 版本兼容性进行评分 (禁用 TLS 1.0/1.1)。
  - [ ] 增加 SSL Grade 评级系统 (A+ -> F)。

- [ ] **高阶网络拓扑分析**
  - [ ] **BGP/ASN 深度洞察**: 展示服务器所属的具体机房和骨干网信息。
  - [ ] **多地域 Ping**: 利用不同的 Edge PoP 模拟不同地区的响应时延。
